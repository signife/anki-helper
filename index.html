        How do I use this?

        1

        Install the AnkiConnect add-on

        In desktop Anki, open Tools → Add-ons → Get Add-ons, install AnkiConnect, and restart Anki.

        2

        Allow the GitHub Pages origin

        In Anki, open Tools → Add-ons → AnkiConnect → Config and add the following origin to webCorsOriginList. 

        {
  "webCorsOriginList": [
    "http://localhost",
    "https://signife.github.io"
  ]
}
        3

        Restart Anki and test the connection

        Save the config, restart Anki, and click Test connection on this page.

        4

        Create the recommended setup

        Choose the default card mode and card font. The note type is fixed as signife_anki_helper. Then create the setup.

        5

        Add cards

        Paste JSON created by ChatGPT and click Add to Anki.
