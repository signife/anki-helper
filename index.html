        How do I use this?
        
        
      
        

      
        
        
        
          
        1
        
          
        
            
        Install Anki and AnkiConnect
        
            
        Install desktop Anki, then install the AnkiConnect add-on and restart Anki.
        
            
        
              
              
            
        
          
        
        
        

        
        
          
        2
        
          
        
            
        Allow this website in AnkiConnect
        
            
        Open Anki → Tools → Add-ons → AnkiConnect → Config. Add this GitHub Pages origin to webCorsOriginList, save, and restart Anki.
        
            
        
              
              
        {
  "webCorsOriginList": [
    "http://localhost",
    "https://signife.github.io"
  ]
}
        
            
        
          
        
        
        

        
        
          
        3
        
          
        
            
        Install the speech engine
        
            
        Install the configured speech engine, launch it once, and add the voices you want to use.
        
            
        
              
            
        
          
        
        
        

        
        
          
        4
        
          
        
            
        Allow this website in the speech engine
        
            
        Open the configured speech engine settings and allow https://signife.github.io when required.
        
            
        
              
              
            
        
          
        
        
        

        
        
          
        5
        
          
        
            
        Start Anki and the speech engine
        
            
        When adding cards with audio, keep both desktop Anki and the configured speech engine running. A green check means both are ready.
        
          
        
        
        

        
        
          
        6
        
          
        
            
        Create the recommended Anki setup once
        
            
        Choose the deck, card mode, font, voice, and speed. Press Create recommended Anki setup. It creates the signife_anki_helper note type with 13 fields and its templates.
        
          
        
        
        

        
        
          
        7
        
          
        
            
        Ask GPT for card JSON
        
            
        Ask GPT to return only one card as valid JSON. reading and exampleReadings should contain full hiragana pronunciations so the speech engine does not misread kanji.
        
            
        
              
            
        
          
        
        
        

        
        
          
        8
        
          
        
            
        Paste the JSON and add the card
        
            
        Paste JSON or choose/drop a JSON/TXT file. A file may contain one card, a JSON array, or multiple JSON objects. Press Add to Anki; cards are processed one at a time and failures are listed at the end.
